# 📚 Book Finder Web App

A simple responsive React app that fetches and displays books using the Open Library API.

---
🔗 Live Demo: https://sparkling-druid-a41835.netlify.app/

## 🚀 Features

* Search books (Enter or button)
* Display title, image, author, description, rating
* Loading & error handling
* No results message
* Show More (load more books)
* Responsive design

---

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Fetch API

---

## 🌐 API

Open Library API:

```
https://openlibrary.org/search.json?q={query}
```

---

## ⚙️ Setup

```bash
git clone <repo-link>
cd book-app
npm install
npm run dev
```

Open: [http://localhost:5173](http://localhost:5173)
